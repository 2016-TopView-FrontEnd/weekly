# css语法

## hsla

 1. H:
    Hue(色调)。0(或360)表示红色，120表示绿色，240表示蓝色，也可取其他数值来指定颜色。取值为：0 - 360
 2. S：
Saturation(饱和度)。取值为：0.0% - 100.0%
 3. L：
Lightness(亮度)。取值为：0.0% - 100.0%
 4. A：
Alpha透明度。取值0~1之间。
